# Space Invader 🚀

A 2D space shooter game built with Unity.

## Gameplay
- Shoot enemies to earn points
- Avoid enemy bullets or lose lives
- Destroy all enemies to win

## Features
- Auto-shooting player
- Enemy AI movement
- Score and lives system
- BGM and sound effects
- Win/Game Over screens

## Built With
- Unity 6
- C#

## How to Play
1. Download the latest release
2. Extract the ZIP
3. Run `Space Invader.exe`
